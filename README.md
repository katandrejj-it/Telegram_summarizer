# Telegram_summarizer

Telegram AI digest: collects messages from your monitored chats/channels via Telethon
(reads as your account), summarizes each chat through the Groq API, and sends a daily
digest to you via your own Telegram bot.

The actual project lives in [`tg_digest/`](tg_digest/). See
[`tg_digest/README.md`](tg_digest/README.md) for setup, configuration and usage.
