# Инструкция по внедрению улучшений v2.0

## Что было изменено

### 1. Новые файлы
- `message_filter.py` — фильтрация сообщений по ключевым словам и умная выборка

### 2. Обновленные файлы
- `config.py` — новый промпт с JSON-структурой, ключевые слова для фильтрации
- `summarizer.py` — парсинг JSON, интеграция фильтрации, улучшенная обработка

## Ключевые улучшения

### ✅ Тематическая группировка
Теперь каждый чат разбивается на отдельные темы с:
- Названием темы
- Кратким описанием
- Ссылкой на начало обсуждения
- Отметкой важности (⚡ важно / 🔹 обычно)

### ✅ Фильтрация контента
- **Исключаются:** война, военные действия, политика
- **Приоритет:** технологии, AI, финансы, эмиграция, подписки

### ✅ Умная обработка длинных чатов
- Вместо простой обрезки — выборка важных сообщений
- Сохраняются начало и конец обсуждений
- Приоритет сообщениям с ключевыми словами

### ✅ Улучшенный формат вывода
```
💬 Название чата
📊 X сообщений

⚡ Важная тема
   Описание обсуждения
   → Перейти к обсуждению

🔹 Обычная тема
   Описание
   → Перейти к обсуждению
```

## Настройка фильтрации

### Режимы фильтрации (config.py)
```python
FILTER_MODE = "soft"   # Рекомендуется: помечает, но не удаляет
FILTER_MODE = "strict" # Полностью удаляет нежелательные сообщения
FILTER_MODE = "off"    # Отключает фильтрацию
```

### Добавление ключевых слов
Отредактируйте `config.py`:

```python
PRIORITY_KEYWORDS = {
    "tech": ["ai", "ml", "gpt", ...],  # Добавьте свои слова
    "finance": ["крипт", "bitcoin", ...],
    # Добавьте новую категорию:
    "travel": ["билет", "отель", "виза"],
}

EXCLUDE_KEYWORDS = [
    "война", "военн", ...  # Добавьте слова для исключения
]
```

## Возможные проблемы и решения

### Проблема: LLM не возвращает JSON
**Симптом:** В логах `Could not parse JSON from response`
**Решение:** Используется автоматический fallback, но можно:
1. Увеличить `temperature` в summarizer.py (сейчас 0.2)
2. Добавить примеры в промпт

### Проблема: Слишком много тем отфильтровано
**Симптом:** Пустые дайджесты или мало контента
**Решение:** 
1. Переключить `FILTER_MODE = "soft"` вместо "strict"
2. Уменьшить список `EXCLUDE_KEYWORDS`
3. Добавить больше `PRIORITY_KEYWORDS`

### Проблема: Неправильные ссылки на темы
**Симптом:** Ссылка ведет не на начало обсуждения
**Решение:** LLM может неправильно определить first_message_id
- Это нормально для первых запусков
- LLM учится на контексте, со временем улучшится

### Проблема: Ошибка импорта message_filter
**Симптом:** `ModuleNotFoundError: No module named 'tg_digest.message_filter'`
**Решение:** Убедитесь, что файл создан:
```bash
ls -la tg_digest/message_filter.py
```
