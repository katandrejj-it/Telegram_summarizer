## 🎯 Ключевые возможности

### 1. Тематическая группировка
Сообщения автоматически группируются по темам с помощью LLM (Groq API).

### 2. Фильтрация контента
- **Исключаются:** война, политика, флуд
- **Приоритет:** технологии, AI, финансы, эмиграция, подписки

### 3. Умная обработка
- Для длинных чатов — умная выборка важных сообщений
- Сохраняются начало и конец обсуждений
- Приоритет сообщениям с ключевыми словами

### 4. Прямые ссылки
Каждая тема содержит ссылку на начало обсуждения в Telegram.

### 5. Отметки важности
- ⚡ — важные темы (решения, факты, полезная информация)
- 🔹 — обычные темы

## 🐛 Решение проблем

### Нет сообщений в дайджесте
```python
# В config.py:
MIN_MESSAGES = 1
FILTER_MODE = "off"
```

### Слишком много тем
```python
# В config.py:
MAX_TOPICS_PER_CHAT = 3
MIN_MESSAGES_PER_TOPIC = 5
```

### Ошибки парсинга JSON
Это нормально, используется автоматический fallback.

Подробнее см. **CHECKLIST.md** → раздел "Возможные проблемы"
