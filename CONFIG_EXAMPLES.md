## Настройка режимов фильтрации

### FILTER_MODE = "off"
- Фильтрация отключена
- Все сообщения обрабатываются
- Используйте для тестирования или если хотите видеть все

### FILTER_MODE = "soft" (рекомендуется)
- Нежелательные сообщения помечаются, но не удаляются
- Приоритет отдается интересным темам
- Баланс между полнотой и релевантностью

### FILTER_MODE = "strict"
- Нежелательные сообщения полностью удаляются
- Только приоритетные темы
- Максимальная фокусировка

## Дополнительные настройки

### Контроль объема дайджеста

```python
# Минимум сообщений для включения чата в дайджест
MIN_MESSAGES = 3  # По умолчанию
MIN_MESSAGES = 10  # Только активные чаты
MIN_MESSAGES = 1  # Все чаты, даже с 1 сообщением

# Максимум тем на чат
MAX_TOPICS_PER_CHAT = 7  # По умолчанию
MAX_TOPICS_PER_CHAT = 3  # Только топ-3 темы
MAX_TOPICS_PER_CHAT = 10  # Больше деталей

# Минимум сообщений для выделения в отдельную тему
MIN_MESSAGES_PER_TOPIC = 3  # По умолчанию
```

### Настройка времени

```python
# В .env файле:
HOURS_BACK=24  # Дайджест за сутки (по умолчанию)
HOURS_BACK=12  # Дайджест за 12 часов
HOURS_BACK=48  # Дайджест за 2 дня

DIGEST_TIME=08:00  # Отправка в 8 утра (по умолчанию)
DIGEST_TIME=20:00  # Отправка в 8 вечера
```

---

## Советы по настройке

1. **Начните с soft режима** — посмотрите что фильтруется
2. **Добавляйте ключевые слова постепенно** — по мере использования
3. **Проверяйте логи** — там видно что отфильтровалось
4. **Корректируйте через 3-5 дней** — после накопления статистики
5. **Используйте --once для тестов** — не ждите scheduler

## Быстрое переключение конфигураций

Создайте несколько файлов конфигурации:

```bash
# Сохраните текущую
cp tg_digest/config.py tg_digest/config_default.py

# Создайте специализированные
cp tg_digest/config.py tg_digest/config_tech.py
cp tg_digest/config.py tg_digest/config_relocation.py

# Переключайтесь по необходимости
cp tg_digest/config_tech.py tg_digest/config.py
```
